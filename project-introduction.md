# Project Introduction

## Main Features

- 博文
  - 发布，查看，修改，删除博文
  - 转发，评论，点赞博文
  - 搜索博文
  - 举报博文，设置博文可见范围
- 话题讨论
  - 话题热榜
  - 话题分类
  - 推荐（话题屏蔽）
- 朋友圈
  - 关注
  - 私信
  - 分组管理
  - 屏蔽
- 拓展
  - 统计粉丝信息
  - 直播
  - 广告

## Tech Stack

- Front-end
  - React
  - Material-UI
- Back-end
  - Spring Boot, Maven
  - JWT
  - Mysql, Redis
- Devops
  - Docker
  - Aws Ec2
  - Aws Amplify
